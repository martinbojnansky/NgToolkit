What's new?

Review the release notes and check out what's new in NgToolkit.

Version 0.13.1

05. Mar 2022

  • Angular@13 peer dependency.
  • Removes deprecated MVVMModule and ObservableStore.
  • Removes obsoleted docs.

Version 0.5.1

01. Dec 2021

  • Adds Enabled directive for dynamic updates of FormControl.enabled property.

Version 0.4.10

20. Aug 2021

  • Adds StopPropagationDirective and PreventDefaultDirective.

Version 0.4.7

18. Aug 2021

  • Changes RxJs import in order to avoid optimization bailouts.

Version 0.4.6

03. Aug 2021

  • Fixes default values of SettingsService-s.
  • Adds Nullable type for strictly typed apps.

Version 0.4.4

26. May 2021

  • @View does not require public ChangeDetectorRef and can be named freely.

Version 0.4.2

19. May 2021

  • Adds @Service, @Query and @View decorators.

Version 0.3.9

17. May 2021

  • Makes @query decorator configurable.
  • Exports ReadonlyBehaviorSubject which ommits next function.
  • Adds computed(...) function that converts anonymouse observables to ReadonlyBehaviorSubject.

Version 0.3.8

10. May 2021

  • Adds StorageModule used to interact with local and session storage.
  • Adds JsonModule used to serialize/deserialize JSON with string date reviver.

Version 0.3.6

7. April 2021

  • Package can be from now on included in the Angular 11.2. application or higher.
