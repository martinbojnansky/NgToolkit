Revives string dates and enables to deep clone items.

JsonModule

and configure it according to your needs by providing a JsonConfig.

JsonConfig

Used to define module behaviour.

Properties

Name Type Default Description
dateFormat RegExp {{ snippets.defaultDateFormat }} Regex used to recognize string date that should be revived.
interceptor boolean false If set to true, all HTTP call will be intercepted and string dates revived.

JsonService

Used to stringify, parse or clone objects with date reviving.

Methods

Name Parameters Returns Description
serialize obj: T T Serializes object into JSON.
deserialize json: string T Deserialized JSON into object and revives dates.
deepClone obj: T T Provides a deep clone by serializing and deserializing the object.
